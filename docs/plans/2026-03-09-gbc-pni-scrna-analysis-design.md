# 胆囊癌单细胞项目分析蓝图（方案三：双轨并行型）

> 最后更新：2026-03-09（经 brainstorming 全面修订）

---

## 1. 项目目标与核心定位

### 1.1 科学定位

**双向对话型**：肿瘤细胞与TME（CAF / 髓系 / 免疫 / 神经相关细胞）之间的相互信号放大了PNI程度。核心输出不是"哪些细胞多了"，而是**哪些配体-受体轴随PNI梯度增强，且双方细胞都有对应的功能状态改变**。

### 1.2 统计策略

scRNA-seq 为**假设生成工具**，验证压力转移至：
- **100例蛋白组队列**（统计验证，n=100足以做生存分析和分组比较）
- **CODEX空间蛋白质组**（空间共定位验证）

所有 scRNA-seq 层结论表述为 `PNI-associated trend / candidate`，不做因果推断。

### 1.3 投稿目标

独立文章，目标期刊：Journal of Hepatology / Clinical Cancer Research / Theranostics 档次。

### 1.4 论文叙事逻辑

```
单细胞发现PNI相关TME重塑机制
    → 100例蛋白组大队列验证临床意义
        → CODEX空间蛋白质组验证空间共定位逻辑
```

## 4. 轨道A：scRNA-seq 分析详设

### 4.1 层1：建图（What exists）

**质控标准：**
- mito% < 25%
- nGene：200 ~ 上四分位数异常值（自适应）
- 双细胞去除：DoubletFinder
- 红细胞/应激细胞过滤

**整合流程：**
```
每样本独立Seurat标准化
  → Harmony批次校正（批次变量：patient_id）
  → PCA（30 PC）→ UMAP
  → Leiden聚类（分辨率0.3起步）
  → 一级注释（10大类）→ 关键群重聚类（二级注释）
  → inferCNV区分恶性上皮 vs 正常上皮
```

**关键风险处理：**
- Schwann细胞数量不足 → 降级至恶性上皮neural program分析
- 癌旁上皮恶性污染 → inferCNV覆盖癌旁上皮群

**一级注释细胞类型（10大类）：**

| 细胞类型 | 核心Marker |
|---------|-----------|
| 上皮/恶性细胞 | EPCAM, KRT8, KRT18, KRT19, MSLN, MUC1 |
| CD8+ T细胞 | CD8A, CD8B, CD3D |
| CD4+ T细胞 | CD4, CD3D, IL7R |
| NK细胞 | GNLY, NKG7, KLRD1 |
| B细胞/浆细胞 | CD79A, MS4A1, MZB1, JCHAIN |
| 巨噬细胞/单核细胞 | CD68, CD14, LYZ, CSF1R |
| 树突状细胞 | CLEC9A, CD1C, FCER1A |
| 中性粒细胞 | FCGR3B, CXCR2, S100A8 |
| 成纤维细胞/CAF | FAP, COL1A1, ACTA2, DCN |
| 内皮细胞/周细胞 | PECAM1, VWF, CDH5, RGS5 |
| Schwann/神经相关 | S100B, MPZ, PLP1, SOX10, MBP |

### 4.2 层2：比较（What changes）

**2A — 细胞比例分析：**

| 比较维度 | 方法 | 注意事项 |
|---------|------|---------|
| PNI趋势（肿瘤内） | 样本级比例图+趋势线 | 仅3个点，不做参数检验，只报告方向 |
| 癌 vs 癌旁（配对） | 配对折线图（per patient） | 强调每位患者内部变化 |
| 双维度交叉 | PNI × 组织类型热图 | 区分"肿瘤共有改变"和"PNI特异改变" |

**2B — cNMF肿瘤程序分析（新增）：**
```
恶性上皮亚群 → cNMF（K=2~8，选最优K）
  → 识别患者共享基因模块
  → 各模块分与PNI趋势关联
  → 与已知程序（EMT/轴突发生/EGFR）对照注释
```

**2C — UCell功能状态评分：**

| 细胞类型 | 评分模块 |
|---------|---------|
| 恶性上皮 | EMT / 轴突发生 / EGFR-AREG轴 / ECM重塑 / 增殖 |
| CAF | mCAF程序 / iCAF程序 / ECM纤维化 / 血管生成支持 |
| 巨噬细胞 | M1样炎性 / M2样免疫抑制 / SPP1+程序 |
| CD8+ T | 细胞毒性 / 耗竭程序 |
| Schwann | 神经髓鞘 / 促侵袭支持（若细胞数足够）|

**2D — 差异表达（Pseudo-bulk DESeq2）：**
- 聚合单位：每位患者 × 每种细胞类型
- 主要对比：
  - 肿瘤 vs 癌旁（per cell type）→ 定义GBC共有改变
  - PNI-High肿瘤 vs PNI-Low肿瘤（per cell type）→ 定义PNI相关改变

### 4.3 层3：机制（Why / How）

**3A — Monocle3轨迹分析（新增）：**
- CD8+ T细胞：初始→效应→耗竭轨迹，PNI-H vs L细胞分布
- 巨噬细胞：M1样→M2样极化轨迹，PNI-H vs L细胞分布

**3B — CellChat全景互作：**
```
全量运行 → PNI-High vs PNI-Low差异比较
重点发送-接收组合：
  malignant ↔ CAF
  malignant ↔ myeloid
  malignant ↔ Schwann
  CAF ↔ myeloid
重点候选轴：
  AREG-EGFR / CXCL/CXCR / LAMB3-DAG1
  SPP1-CD44 / VEGFB-FLT1 / TGFβ-TGFBR
  轴突引导相关（NRP1-SEMA / L1CAM）
```

**3C — pySCENIC转录调控网络（新增）：**
- 分析对象：恶性上皮 + CAF + 髓系
- 重点TF候选：STAT3、EGR1、SOX10、ZEB1、NF-κB
- 输出：PNI-High特异性激活TF及其靶基因网络

**3D — 候选轴优先级打分（输出清单A）：**

每条候选互作轴按以下4项打分（各1分）：

| 评分项 | 说明 |
|--------|------|
| PNI趋势一致 | PNI-H > M > L方向明确 |
| 肿瘤 > 癌旁 | 在肿瘤中信号更强 |
| 文献支持 | 参考文献中有相关报道 |
| 蛋白可检测 | 有商业抗体，适合CODEX |

≥3分进入高优先级候选清单A。

## 5. 轨道B：蛋白组整合详设

### 5.1 已有结论结构化整理

整理格式：
```
蛋白名称 | UniProt ID | PNI相关方向(↑/↓) | 差异显著性 |
所在通路 | 已知细胞来源 | 证据类型
```

### 5.2 补充分析（若蛋白组未做PNI分析）
```
100例蛋白表达矩阵
  → PNI分组差异分析（limma + BH校正）
  → GSEA富集（Hallmark + KEGG + GO）
  → Cox回归 + Kaplan-Meier生存分析
  → 输出候选清单B
```

重点关注通路：ECM重塑 / 轴突发生 / IL6-STAT3 / EGFR-MAPK / 免疫检查点

### 5.3 双轨交集筛选标准

候选进入高优先级（满足≥2条）：
- ✓ scRNA-seq中PNI趋势方向明确
- ✓ 蛋白组中PNI分组差异显著（adj.p < 0.05）
- ✓ 蛋白组中与生存预后相关（Cox p < 0.05）
- ✓ CellChat中对应通路增强
- ✓ 文献有报道
- ✓ 蛋白有商业抗体

| 等级 | 满足条件数 | 处理策略 |
|------|-----------|---------|
| 一级 | ≥4条 | 进入CODEX panel核心槽位 |
| 二级 | 3条 | 按剩余槽位纳入 |
| 三级 | 2条 | 备选 |
| 排除 | ≤1条 | 探索性记录 |

### 5.4 蛋白组→scRNA-seq 反向指导

蛋白组强PNI相关蛋白，在scRNA-seq中溯源：
- 哪类细胞高表达？（细胞来源解析）
- 哪种细胞状态下表达最高？
- 与哪些配体/受体共出现？（互作上下文）

## 6. CODEX 空间验证详设

### 6.1 Panel 结构（~50蛋白）

| 槽位 | 蛋白数 | 来源 |
|------|--------|------|
| 细胞类型骨架 | ~20 | 固定，见下表 |
| 功能状态 | ~15 | 一级候选（≥4条筛选标准）|
| 候选互作蛋白 | ~10 | CellChat高优先级配体/受体 |
| 组织结构 | ~5 | 神经+血管+基底膜 |

**细胞类型骨架（固定）：**

| 细胞类型 | 推荐蛋白 |
|---------|---------|
| 恶性上皮 | EPCAM、KRT19、MUC1 |
| CAF | FAP、ACTA2、COL1A1 |
| 巨噬细胞 | CD68、CD163、SPP1 |
| CD8+ T | CD8A、PD-1、TIM-3 |
| CD4+ T/Treg | CD4、FOXP3 |
| B/浆细胞 | CD79A、MZB1 |
| 内皮细胞 | CD31、LYVE1 |
| Schwann/神经 | S100B、SOX10 |
| 组织结构（必选）| **PGP9.5（神经轴突）**、SMA、ColIV |

> PGP9.5是神经纤维染色金标准，用于精确标定PNI灶位置，作为所有空间分析的锚点。

### 6.2 空间分析模块

**模块一：神经周细胞密度梯度**
```
PGP9.5标定神经纤维锚点
  → 计算各细胞类型在0-50μm / 50-100μm / >100μm的密度
  → 比较PNI-High vs PNI-Low密度梯度差异
```

**模块二：空间邻域（Niche）分析**
- 工具：SCIMAP 或 Squidpy
- 每个细胞计算10个最近邻组成 → K-means聚类识别3-6个空间niche
- 比较PNI-High vs PNI-Low中各niche面积比例
- 对标Paper 2的niche类型：神经周微龛 / 纤维炎性微龛 / 免疫抑制微龛 / 血管神经邻近区

**模块三：配体-受体空间共定位**
- 从CellChat高优先级轴中选3-5条
- 验证配体细胞与受体细胞空间距离（<30μm定义为接触）
- 比较PNI-High vs PNI-Low共定位频率

**候选验证轴：**

| 轴 | 配体细胞 | 受体细胞 | 验证蛋白对 |
|----|---------|---------|-----------|
| LAMB3-DAG1 | 恶性上皮 | Schwann | LAMB3 + DAG1 |
| SPP1-CD44 | SPP1+TAM | 恶性上皮 | SPP1 + CD44 |
| AREG-EGFR | 髓系/CAF | 恶性上皮 | AREG + EGFR |
| VEGFB-FLT1 | iCAF | 内皮细胞 | VEGFB + FLT1 |

### 6.3 CODEX → scRNA-seq 反向诠释
```
CODEX发现的空间niche组成
  → scRNA-seq中对应亚群的PNI趋势
  → CellChat中对应互作的信号强度
  → 形成空间-分子闭环证据链
```

## 8. 统计与解释原则

- 优先采用：配对比较 / 样本级可视化 / 伪bulk / 趋势分析

- 推荐表述：`PNI-associated remodeling` / `candidate mechanism` / `spatially testable hypothesis`

## 11. 关键风险与降级策略

| 风险 | 触发条件 | 降级策略 |
|------|---------|---------|
| Schwann细胞捕获不足 | 每样本<50个Schwann细胞 | 转向恶性上皮neural program + 配体受体证据 |
| cNMF模块无PNI趋势 | 所有模块无明确梯度 | 退回UCell预设程序打分为主 |
| CellChat信号噪声高 | 候选轴在癌旁也强 | 增加肿瘤特异性过滤条件 |
| CODEX panel槽位不足 | 候选蛋白>60个 | 按优先级等级严格筛选，一级必选 |
| 蛋白组PNI分组无显著差异 | adj.p全部>0.05 | 改用连续PNI评分（若有）或方向一致性替代 |
