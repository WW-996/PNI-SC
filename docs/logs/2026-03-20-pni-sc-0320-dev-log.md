- 避免 pandas 默认索引顺序导致图形左右位置不稳定。

### 7. 大类 marker dotplot 对齐修复

- 用户更新 `broad_celltype` 标签后，发现 marker dotplot 结果与新注释不再对应。
- 根因是 marker 字典键名与实际 broad annotation 标签名不再完全一致，例如：
  - `plasma cell`
  - `plasma_cell`
- 修复方式：
  - 增加 `canonical_label()` 规范化函数
  - 将 marker 字典标签与当前 `broad_celltype` 实际分类标签自动对齐
  - 基于对齐后的 `aligned_markers` 作图
  - 同时重建 `broad_celltype` 的分类顺序，保证 dotplot 行列逻辑稳定
